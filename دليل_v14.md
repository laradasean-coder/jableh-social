# دليل الإصدار 14 — المساعد الذكي + التحويل + الأرشفة

## ما الجديد

### 1. مساعد ذكي حقيقي بـ DeepSeek
- `supabase/functions/ai-assistant/` يحلّل أسئلتك بالعربية ويجيب من بيانات الدائرة الحية.
- زر عائم أخضر يفتح نافذة محادثة مع المساعد (ليست غرفة محادثات بشرية).

**التفعيل:**
```bash
supabase secrets set DEEPSEEK_API_KEY=sk-xxxxx
supabase functions deploy ai-assistant
```
احصل على المفتاح من platform.deepseek.com

### 2. تحويل المستفيد لجمعية بضغطة
- زر "تحويل لجمعية" في صفحة تفاصيل المستفيد.
- يُسجّل التحويل في جدول `referrals` وتراه الجمعية المعنية.

### 3. الأرشفة
- زر "أرشفة" مع سبب (تحسّن الوضع، انتقال، وفاة...).
- المؤرشف يختفي من القوائم النشطة، وزر "إلغاء الأرشفة" يسترجعه.

### 4. كل شيء أونلاين — صفر تخزين محلي
- تفضيل الوضع الليلي انتقل لملف المستخدم في قاعدة البيانات (`profiles.theme`).
- لا `localStorage` ولا `sessionStorage` في كامل المشروع.

## النشر (بالترتيب)
1. شغّل migrations الجديدة: `011_referrals_archive.sql` (و009 المُحدّث به عمود theme)
2. فعّل DeepSeek كما أعلاه
3. ارفع على Vercel
